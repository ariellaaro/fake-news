# A comparative analysis of fake and verified news dissemination

This repository contains the resources for the associated paper, including:

- The full MC Fake dataset (originally available [here](https://github.com/qwerfdsaplking/MC-Fake));
- Annotated scripts:
  - `metrics.py` - network metrics and community detection statistics;
  - `SIR_models.py` - dual SIR (Susceptible–Infected–Recovered) model and simulations;
- High-resolution versions of the images presented in the paper.
